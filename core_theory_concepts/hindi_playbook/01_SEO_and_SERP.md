# SEO Aur SERP Kya Hai? (Simple Notes)

## 1. SEO Kya Hai? (Search Engine Optimization)
Bhai simple words mein, SEO ek aisi technique hai jisse hum apni website ko Google pe number 1 ya top par laate hain. 
Agar mera ek blog ya website hai, aur main chahta hu ki log jab kuch search karein toh meri website sabse upar dikhe, tab mujhe SEO karna padega.

*   **Basically:** Apni website ko Google ke rules (algorithm) ke according set karna taki free (organic) traffic aaye.
*   **Kyu zaruri hai?:** Kyunki log search karne ke baad second page par nahi jate! Sab peli page ke links pe click karte hain.

## 2. SERP Kya Hai? (Search Engine Results Page)
SERP ka matlab wahi page hai jo Google (ya kisi bhi search engine) par kuch search karne ke baad samne khulta hai.
Jaise agar maine search kiya `best shoes`, toh jo 10 links aur results ayenge, us poore page ko SERP bolte hain.

*   **Simple meaning:** Google ka result page jahan humari website rank karti hai.
