# Topic 9: Keywords Kya Hote Hain, Inki Types aur Keyword Research

---

## 1. Keywords Kya Hain?

Keywords wo words ya phrases (jumlay) hote hain jo log Google ya kisi bhi search engine par type karte hain kuch dhoondne ke liye.

> **Simple Example:**
> Agar aapko naye joote khareedne hain toh aap Google par type karenge *"men's running shoes"* — yahi ek **keyword** hai. SEO ka kaam yeh hai ke jab koi yeh type kare, toh aapki website sabse pehle nazar aaye.

---

## 2. Keywords Ki Types

Keywords ko unki **length** aur **user ke iraade** ke hisaab se divide kiya jaata hai.

---

### A. Length ke Hisaab Se

#### Short-tail Keywords
- **Hote kya hain:** 1 ya 2 words ke keywords
- **Search Volume:** Bahut zyada (lakho log search karte hain)
- **Competition:** Bahut zyada — rank karna mushkil
- **Misal:** `"Shoes"`, `"SEO"`, `"Laptop"`

> **Agar sirf inhi par focus kiya toh:** Naye websites ka in par rank karna almost impossible hota hai — Google par pehle se bade brands baithein hain.

---

#### Long-tail Keywords
- **Hote kya hain:** 3 ya 3 se zyada words ke keywords
- **Search Volume:** Kam — lekin **zyada targeted**
- **Competition:** Kam — rank karna asaan
- **Misal:** `"Men's Nike running shoes size 10"`, `"Technical SEO checklist for beginners"`

> **Tip:** Nayi website ke liye hamesha long-tail keywords se shuru karein. Traffic thoda kam hoga lekin jo aayega woh **convert** karega — kyunke user exact cheez dhoondh raha hota hai.

---

#### LSI (Latent Semantic Indexing) Keywords
- **Hote kya hain:** Aapke main keyword se milte julte, related words
- **Kaam kya karte hain:** Google ko samajhne mein madad karte hain ke aapka page asal mein kis topic par hai
- **Misal:** Agar main keyword `"Apple"` hai, toh LSI keywords honge: `iPhone`, `MacBook`, `App Store`, `Tim Cook`

> **Agar nahi use kiya toh:** Google confuse ho sakta hai — "Apple" fruit ke baare mein hai ya company ke baare mein? LSI keywords yeh ambiguity khatam karte hain.

---

### B. User Intent ke Hisaab Se

User jo search karta hai, uske peeche koi **iraada (intent)** hota hai. Is intent ko samajhna SEO mein sabse zaroori cheez hai:

| Intent Type | User Kya Chahta Hai | Misal |
|-------------|---------------------|-------|
| **Informational** | Kuch seekhna ya jaanna | `"how to bake a cake"` |
| **Navigational** | Kisi specific website par jaana | `"Facebook login"` |
| **Commercial** | Compare karna, research karna | `"best laptops 2025"` |
| **Transactional** | Abhi kharidna chahta hai | `"buy iPhone 15 Pro Max"` |

> **Zaroori Baat:** Agar aap ek e-commerce site chalate hain toh **Transactional** aur **Commercial** keywords aapke liye sabse zyada valuable hain. Blog ke liye **Informational** keywords best hote hain.

## 3. Keyword Research Kaise Karein?

Yeh ek step-by-step process hai jo aapko sahi keywords dhoondne mein madad karta hai:

**Step 1 — Brainstorming**
Sochiye ke aapka ideal user kya type karega. Apne aap se poochhein: *"Agar mujhe yeh cheez dhoondni hoti toh main kya search karta?"*

**Step 2 — Free Tools Se Shuru Karein**
- **Google Keyword Planner** — bilkul free, Google ka apna tool
- **Ubersuggest** — beginners ke liye best
- **Google Search ka "People Also Ask"** — goldmine hai, ignore mat karein

**Step 3 — Paid Tools (Jab Budget Ho)**
- **Ahrefs** — industry ka best tool
- **SEMrush** — competitor analysis ke liye zabardast

**Step 4 — Competitors Ki Jasoosi**
Apne top competitors ki website par jaayein aur dekhen ke woh kin topics par likh rahe hain. Unke headings, meta titles padhein — wahan best keywords chuppe hote hain.

**Step 5 — Sahi Mix Banayein**
Sirf ek type par mat atko. Ek acha keyword strategy mein hona chahiye:
- 2-3 Short-tail keywords (long-term goals ke liye)
- 8-10 Long-tail keywords (quick wins ke liye)
- LSI keywords naturally content mein

## 4. AI Se Perfect Keywords Nikaalein

Sabse fast aur accurate keyword research ab AI ke zariye ho sakti hai — aapko bas apni website ka code dena hai.

**Best Platforms:**
- **Claude** — claude.ai
- **LMSYS Chatbot Arena** — lmarena.ai (yahan Claude Opus ya GPT-4o select kar sakte hain)
- **Gemini Advanced** — gemini.google.com

**Karna Kya Hai:**
Apni `index.html` file ka pura code copy karein aur neeche diya gaya prompt AI mein paste karein. Website explain karne ki zaroorat nahi — AI khud code padhke samajh jaayega.

**AI Keyword Research Prompt:**

```
Act as a world-class SEO strategist and Keyword Research expert.

Below is the full HTML code of my website's homepage (index.html).

Analyze the structure, text, meta tags, and overall context to understand exactly what my business is about.

Based on this analysis, provide a complete keyword strategy categorized as:

1. Primary Keywords — Short-tail, high volume (2-3 keywords)
2. Long-Tail Keywords — Low competition, high conversion (8-10 keywords)
3. LSI / Semantically Related Keywords — To build topical authority (5-7 keywords)
4. Intent-Based Keywords — Grouped by:
   - Informational (user seeking knowledge)
   - Commercial (user comparing options)
   - Transactional (user ready to buy)

For each keyword, briefly explain WHY it fits my specific website.

Return results in a clean table format with columns:
Keyword | Type | Search Intent | Difficulty (Est.) | Why It Fits

Here is my index.html code:
[YAHAN APNA INDEX.HTML KA PURA CODE PASTE KAREIN]
```
