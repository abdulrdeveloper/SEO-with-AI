# Search Engine Kya Hai aur Kaise Kaam Karta Hai?

## 1. Search Engine Kya Hai?
Search engine internet ka ek bohot bada software tool hai. Jab hum is par koi sawal type karke puchte hain, toh yeh internet ki karodo websites mein se sabse sahi jawab dhoondh kar hamare samne rakh deta hai.

## 2. Search Engine Kaam Kaise Karta Hai?
Iske mainly 4 simple steps hote hain:
1. **Crawling (Dhoondhna):** Search engine internet par apne "bots" ya "spiders" bhejta hai jo naye-naye web pages ko dhoondhte hain aur unko scan karte hain (padhte hain).
2. **Indexing (Store Karna):** Jo bhi nayi information bots ko milti hai, search engine usko apne huge database (index) mein save kar leta hai. Bilkul waise jaise kisi library mein nayi books ko category ke hisab se shelf pe lagaya jata hai.
3. **Ranking (Position Tay Karna):** Jab aap aakar kuch search karte hain, toh search engine apne is saved index mein se sabse best aur relevant results chunta hai aur unki ek list tayar karta hai (No. 1, 2, 3... ke hisab se).
4. **Displaying Results (SERP Dikhana):** Akhir mein, us ranking ki list ko hamari mobile/computer screen par dikhata hai, aur isi dikhne wale page ko hum SERP (Search Engine Results Page) kehte hain.

## 3. Duniya Bhar Ke Popular Search Engines
Aisa nahi hai ki poori duniya mein sirf Google hi chalta hai. Alag-alag countries mein alag search engines use hote hain:
*   **Duniya ka Boss:** **Google** (Puri duniya mein 90% se zyada log yahi use karte hain).
*   **Microsoft Ka Apna:** **Bing** (Yeh Windows PC ke sath aata hai aur dusre number par hai).
*   **Privacy Ke Liye:** **DuckDuckGo** (Log ise isliye pasand karte hain kyu ki yeh user ka data track nahi karta).
*   **China Mein:** **Baidu** (Kyunki China mein Google banned hai, toh wahan sab Baidu use karte hain).
*   **Russia Mein:** **Yandex** (Russia mein logon ka most favorite search engine).
*   **South Korea Mein:** **Naver** (Inki majority janta Naver par search karti hai).
